📦 Mes Boîtes:

Python

Python est un langage de programmation open source, polyvalent et très populaire, créé en 1989 par Guido van Rossum.

HTML

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour décrire la structure et le contenu des documents affichés sur le web.

CSS

Le CSS (Feuilles de style en cascade) est un langage utilisé pour décrire la présentation et la mise en page d'un document structuré (comme HTML ou XML).

C

Le langage C est un langage de programmation de bas niveau, conçu pour être proche de la machine tout en offrant un certain niveau d'abstraction, permettant une communication directe avec la mémoire, le processeur et les périphériques.

JavaScript

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives et à ce titre est une partie essentielle des applications web.

Linux

Linux (parfois nommé GNU/Linux) est un système d'exploitation open source de type Unix, basé sur le noyau Linux créé en 1991 par Linus Torvalds.
